untitled budget app is a full stack web app that replaces my clunky google sheets tracking

it was built with [sveltekit](https://svelte.dev/) and [tailwindcss](https://tailwindcss.com/)

to use the app for yourself

1. clone the repo
2. `npm install`
3. `npm run dev`
